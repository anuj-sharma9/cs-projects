This objective of this game is to get SpongeBob catch the jellyfish (the magenta square)
in order to win the game. If spongebob goes through and touches the sea urchins (the black squares),
then the player loses the game. The basic controls to control SpongeBob is the arrow keys on the keyboard.
The game starts in the Start page where the user must hit ENTER to begin the game. At any stage in the game, 
if the user hits BACKSPACE, the user will be returned to the start page of the game.
